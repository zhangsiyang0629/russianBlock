# 俄罗斯方块微信小游戏界面布局规格

## 设备方向
- 竖屏（Portrait）

## 整体布局分区
屏幕高度按比例划分为三个主要区域，从上至下：

| 区域          | 高度占比 | 说明                                     |
|---------------|----------|------------------------------------------|
| 顶部广告位    | 1/9      | 预留广告展示区域，不遮挡游戏核心操作     |
| 游戏主区域    | 剩余部分 | 包含游戏画布、分数、预览等核心游戏内容   |
| 底部按键区    | 1/5      | 固定操作按钮栏，用于控制方块             |

## 游戏主区域内元素
- **暂停按钮**：位于游戏主区域的**右上角**，用于暂停/继续游戏。

## 底部按键区详细规格

按键从左至右等距排列，水平居中于底部安全区域内。

### 按键 1：左移动键
- **样式**：类似左箭头 `←` 的图标按钮
- **功能**：将当前下落方块向左移动一格

### 按键 2：向下加速键
- **样式**：圆形按钮，内部包含向下的箭头符号 `↓`
- **功能**：加快当前方块下落速度（软降）

### 按键 3：变换键
- **样式**：圆形按钮，内部包含随机/洗牌风格的符号（如 `⟳` 或 `🔄`）
- **功能**：旋转/变换当前方块的朝向

### 按键 4：右移动键
- **样式**：类似右箭头 `→` 的图标按钮
- **功能**：将当前下落方块向右移动一格

## 交互补充说明
- 底部按键区背景建议半透明深色，确保在各类游戏背景下均清晰可见
- 按钮尺寸需满足微信小游戏可点击区域规范（不小于 44x44 像素逻辑点）
- 广告位不参与游戏交互事件处理，需与游戏画布清晰分离
