# 俄罗斯方块微信小游戏 - 得分表现需求规格

## 概述
本需求描述玩家完成消层后，得分的视觉反馈表现。包括得分飘字的上升淡出动画、星星特效飞入分数面板，以及分数面板的数字跳变效果。不同消层层数对应不同的飘字颜色与大小，旨在强化高难度消除的爽快感。

---

## 触发条件
- 方块落定并完成消层判定后，**对每一次消层事件（可合并多层同时消除的情况）**触发一次得分表现。
- 若一次落定同时消除多行（如2行、3行、4行），**仅生成一个飘字**，显示本次消层的总得分，颜色与大小按消除行数处理。

---

## 飘字特效

### 显示内容
- 飘字文案为本次消层所得分数，例如 `+10`、`+25`、`+50`、`+80`。
- 文案前端带 `+` 号。

### 生成位置
- 飘字生成于**被消除行的中心位置**（若多行消除，取消除范围的行中心与列中心）。

### 颜色规则

| 消除行数 | 飘字颜色 | 色值参考 |
|----------|----------|----------|
| 1 行 | 灰色 | `#999999` |
| 2 行 | 绿色 | `#33CC33` |
| 3 行 | 蓝色 | `#3399FF` |
| 4 行 | 紫色 | `#9933FF`（或 `#CC33FF`） |

### 字体大小规则

| 消除行数 | 字体大小（Canvas px） | 说明 |
|----------|----------------------|------|
| 1 行 | 20 ~ 24 | 基础字号 |
| 2 行 | 22 ~ 26 | 略大于1行 |
| 3 行 | 26 ~ 30 | 明显大于1行 |
| 4 行 | 36 ~ 42 | **显著增大**，突出4行消除的成就感 |

> 具体字号可根据游戏区域尺寸动态适配，上表为 750×1334 设计分辨率下的参考值。

### 动画表现

| 阶段 | 持续时间 | 效果描述 |
|------|----------|----------|
| **上浮** | 0.8 秒 | 飘字从生成位置**匀速上浮**约 40~60 像素（根据游戏区域等比例调整）。 |
| **淡出** | 0.4 秒（与上浮后段重叠） | 飘字在上浮的最后 0.4 秒内透明度从 1.0 过渡至 0.0。 |
| **总时长** | 约 0.8 ~ 1.0 秒 | 淡出结束后飘字完全消失。 |

### 字体样式
- **字体**：粗体，`bold`。
- **描边**：黑色外描边（线宽 2~3px），增强可读性。
- **阴影**（可选）：轻微黑色投影，增加立体感。

---

## 星星特效

### 触发时机
- 飘字**完全淡出消失**后，在原飘字位置生成星星特效。

### 美术资源

| 资源项 | 路径 | 说明 |
|--------|------|------|
| 星星纹理 | `subpackages/images/star.png` | 星星图标，带透明通道 |

### 动画表现

| 阶段 | 持续时间 | 效果描述 |
|------|----------|----------|
| **闪烁出现** | 0.5 秒 | 星星以极小比例（如 0.1 倍）出现，**快速缩放至 1.0 倍**，同时带有**透明度闪烁**（如 0.5 → 1.0 → 0.7 → 1.0 反复），制造闪耀登场的感觉。 |
| **飞向分数面板** | 0.6 秒 | 星星从当前位置**沿弧线或直线**移动至顶部/侧边分数面板的显示位置。建议使用缓出曲线，到达时轻微减速。 |
| **到达后消失** | 瞬间 | 星星抵达分数面板后立即消失（或缩小消失，0.1 秒），随后触发分数面板跳字。 |

### 闪烁实现参考
```javascript
// 闪烁阶段：透明度或缩放周期性波动
const twinkleSpeed = 8; // 每秒闪烁次数
const alpha = 0.5 + 0.5 * Math.abs(Math.sin(elapsed * twinkleSpeed * Math.PI));
const scale = 0.8 + 0.2 * Math.abs(Math.sin(elapsed * twinkleSpeed * Math.PI));
```

## 分数面板跳字

### 触发时机
- 星星抵达分数面板并消失后，立即触发分数面板的数字跳变。

### 跳变表现
- 分数面板的数字从当前值开始，快速滚动递增至新值。
- 滚动时长建议：0.3 ~ 0.5 秒。

---

## 交互影响

| 交互项 | 飘字/星星动画期间行为 |
|----------|----------|
| 方块下落 | 正常进行，不受飘字影响 |
| 方向键 | 正常响应 |
| 暂停按钮 | 若暂停，飘字和星星动画同步暂停，恢复后继续 |

## 连续消层处理

- 若一次方块落定消除多行（如同时消2行），只显示一个飘字，总分开显示（例如 +25），颜色和大小按消除行数处理。
- 若短时间内连续发生多次消层（如连续快速消层），每个消层事件独立生成飘字和星星，不互相取消或合并。
- 如果前一个飘字尚未消失就发生新消层，新的飘字可以叠加显示（例如在更高或偏移一点的位置），互不干扰。
