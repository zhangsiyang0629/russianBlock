# 俄罗斯方块微信小游戏 - 封面特效需求规格

## 概述
本需求描述游戏封面（主界面）的特效表现，包括背景烟花特效与用户点击反馈特效。所有特效需与封面 UI 元素（Play 按钮、排行榜入口、设置入口等）共存，不影响按钮的正常点击交互。

---

## 一、烟花绽放特效

### 触发规则
- **播放频率**：每隔 **1 ~ 3 秒**（随机间隔）自动播放一次烟花绽放特效。
- **播放位置**：在封面可见区域内**随机位置**生成，烟花绽放中心点不应超出屏幕边界。
- **循环模式**：封面展示期间持续循环，进入游戏后停止。

### 特效表现（参考）
- 烟花从中心点向四周扩散，包含粒子飞散、渐隐消失的效果。
- 颜色可随机变化，或从预设的几种颜色（如红、金、蓝、绿）中随机选取。
- 单次烟花特效持续时间建议在 **0.5 ~ 1 秒** 之间。

### 实现建议
- 使用 Canvas 粒子系统实现，或使用 Lottie / 序列帧动画。
- 粒子数量控制在适当范围，避免性能问题。
- 烟花特效层级应**低于**所有可交互按钮，不阻挡点击事件。

---

## 二、点击特效

### 触发规则
- **触发条件**：用户在封面**任意位置**点击（包括空白区域和按钮区域）。
- **频率限制**：两次点击特效之间的最小间隔为 **600ms**。若用户在间隔内连续点击，忽略后续点击，不生成新特效。
- **循环模式**：封面展示期间持续生效，进入游戏后停止。

### 特效表现
- **设计要求**：低调、不夸张，作为轻量级的点击反馈。
- **参考方案**（任选其一或自行设计）：
  - **涟漪效果**：点击处出现一个半透明圆环，向外扩散并逐渐消失，持续时间约 300ms。
  - **小星光**：点击处产生 3~5 个微小粒子，向随机方向飘散并消失。
  - **浅色光点**：点击处短暂出现一个柔和光点，停留约 200ms 后淡出。
- **颜色建议**：使用白色或浅金色，透明度适中，不与封面背景色冲突。

### 实现建议
- 使用 Canvas 绘制，在 `touchstart` 事件中记录点击坐标并生成特效。
- 特效层级应**高于**烟花特效，但**低于**所有按钮元素，或设置为不阻挡事件穿透。

---

## 三、性能与兼容性要求

| 项目 | 要求 |
|------|------|
| 帧率影响 | 特效运行时不应导致游戏封面出现明显卡顿，建议保持在 30fps 以上。 |
| 内存控制 | 粒子系统需及时回收已消失的粒子对象，避免内存泄漏。 |
| 低端机适配 | 在低性能设备上可自动降低粒子数量或特效频率，保证基础体验。 |
| 与按钮交互 | 所有特效**不得拦截或延迟**按钮的点击事件。点击特效需在按钮点击逻辑执行的同时播放。 |

---

## 四、交互说明

- 用户点击按钮时，**同时播放**点击特效，两者互不阻塞。
- 烟花特效在按钮区域上方播放时，按钮仍可正常点击。
- 进入游戏后，封面隐藏，所有封面特效停止。返回封面时，特效重新启动。
